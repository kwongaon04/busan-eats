부산 맛집 데이터 기반 웹 프로젝트

맛집 페이지로 이동

1) 프로젝트 목적

부산 지하철역 주변 음식점 데이터를 공공데이터로 확보하고, 파이썬으로 정제·가공한 뒤 웹페이지에서 조회할 수 있도록 구현한다. “데이터 수집 → 가공 → 웹 적용” 흐름을 경험하는 것이 목표다.

2) 부산과 음식 문화

부산은 바다와 가까워 해산물 음식이 발달했고, 돼지국밥·밀면·어묵 등 지역 대표 음식이 뚜렷하다. 지하철역 주변에 상권이 모이는 특성이 있어 “역 기준 맛집 정보”가 실제 활용에 적합하다.

3) 데이터 처리 흐름

  • 공공데이터 CSV 확보
  • 파이썬으로 필요한 컬럼 추출 및 공백/형식 정리
  • JSON(restaurants.json)으로 변환
  • 웹에서 JSON을 불러와 카드 형태로 출력

4) 기대 효과

  • 데이터 기반 웹서비스의 기본 구조 이해
  • 공공데이터를 “실제 화면”으로 연결하는 경험
  • 추후 지도 API, 리뷰 기능 등 확장 가능
